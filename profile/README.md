# GoSniff

*Go-powered network packet sniffing tools and composable libraries.*

Point it at an interface and let it listen. GoSniff gives you composable building blocks for capturing, filtering, and inspecting network traffic, all in idiomatic Go.
Quiet on the wire. Loud in the logs.

- :bar_chart: Real-time traffic dashboards and export to common analysis formats
